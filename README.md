## Instalação
Atualizando o nodejs

```shell
$ sudo npm install -g n
$ sudo n latest
$ sudo n stable
```

Instalando o adonisjs

```shell
$ npm init adonis-ts-app@latest api-adonisjs
```

Iniciando o server
```shell
$ node ace serve --watch
```

Instalando e configurando o lucid
```shell
$ npm i @adonisjs/lucid
```
